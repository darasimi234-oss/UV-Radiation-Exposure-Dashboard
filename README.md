# UV Radiation Exposure Dashboard

## Project Overview

This project presents an interactive Microsoft Excel dashboard developed to analyze ultraviolet (UV) radiation exposure across three different environments:

- Open Space
- Under Tree Canopy
- Classroom

The dashboard summarizes UV radiation data using charts, key performance indicators (KPIs), and data insights to support the interpretation of exposure patterns and environmental conditions.

## Objectives

- Analyze UV radiation exposure across different environments.
- Compare daily and hourly mean UV exposure.
- Present key environmental statistics using KPI cards.
- Communicate findings through a clear and professional dashboard.

---

## Tools Used

- Microsoft Excel
- Charts
- Pivot Tables
- Data Visualization
- Statistical Analysis (Mean, Maximum, Minimum)

---

## Dashboard Features

- Daily Mean UV Exposure Chart
- Hourly Mean UV Exposure Chart
- Highest UV Exposure KPI
- Lowest UV Exposure KPI
- Mean Temperature KPI
- Mean Humidity KPI
- Key Insights Section

---

## Key Findings

- Open Space recorded the highest UV radiation exposure.
- Under Tree Canopy significantly reduced UV exposure.
- Classroom recorded the lowest UV exposure.
- UV radiation peaked during the afternoon hours.
- Shade and indoor environments effectively reduced UV exposure.
